ubuntu安装

安装的话大致流程就是

在Windows下分盘->制作启动盘->u盘启动安装->下一步下一步下一步（可能分盘会麻烦一点，但是可以只分一个/

安装完之后马上要做的事

  • 网卡 检测网卡是否正常，如果可以直接开始上网，那就是非常的不戳

有可能开机之后没有wifi选项，解决办法是：先连网线或者手机usb数据共享，然后去网上源码编译一下网卡驱动就ok了

但是不排除究极螃蟹网卡（对 我说的是mt7921 会时好时坏）最后我直接换了ax210

  • 换源 选择一个国内源安装软件会更快，在软件更新这个软件里点点就行
  • 显卡驱动 安装显卡驱动（如果有显卡的话），首选方法是在软件更新这个软件里面点一下安装
    • 如果重启电脑正常启动，那恭喜你非常的顺利
    • 如果再次进入ubuntu，就在开机时候选第二个找一个恢复模式启动，再去软件更新里把显卡驱动卸载掉（点到最后一个就是卸载显卡驱动）
    • 如果还是要装显卡驱动可以去官网下载，但如果安装之后还是无法进入ubuntu，则需要开机选第二个然后选第一个再选root进入命令行找到显卡驱动文件执行./xxxxx --uninstall进行卸载

非常好用的工具

  • 谷歌搜索

    谷歌一搜就是官网，就算是广告也有明显标注，百度 懂得都懂

    如果想看中文搜索结果也可以点工具->选择语言

  • telegram

    ubuntu的QQ上一次更新是在2020年，远古版QQ，根本没维护了

    国外一般都用telegram，相当于国内QQ，唯一的不方便之处是必须要梯子

    但真的好看又好用

  • clash for windows

    虽然是for Windows，但是有ubuntu版本，非常的不戳

    clash 懂得都懂

ubuntu上需要在网络中配置中选到手动 设置为127.0.0.1和7890端口

  • proxychains/proxychains4

    在某一个命令前面加上这串，一定会让这个命令走上代理，适用于规则代理或者全局代理不管用的时候

    如果使用的是clash的话，点port那行有个终端图标，复制命令之后运行也是同样的效果

  • scp

    对远端服务器直接进行复制粘贴

    scp -r -P 316 ./xxx root@xx.xx.xx.xx:/home/xx

  • vim/nano

    终端文本编辑器，我用vim

  • tmux/screen

    tmux>screen，可窗口多开，一窗口可分开成多个隔间

  • 输入法

    首推还是搜狗，词库雀食是强大的 如果无法使用可以安装ibus的输出法，一定不会报错

  • ubuntu优化

    战老师yyds！

  • ros（不是

    做机器人相关项目才会需要

    放个xtdrone文档放这：https://www.yuque.com/xtdrone/manual_cn/basic_config#iud2s

    要不什么时候写个教程？

  • 飞书（不是

    先进团队都用飞书（）

  • clion

    sudo snap install clion --classic直接安装，然后输入clion运行即可

    如果是下载的压缩包，则需运行里面的clion.sh脚本

    在.bashrc/.zshrc中写入alias clion=xxx/bin/clion.sh 之后就也可以输入clion命令打开了
